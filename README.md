# MACHBANK UI

> A mobile-first neobanking interface prototype — minimalist, opinionated, and free of visual noise.

## What is this?

A public playground for experimenting with modern banking UX patterns. The thesis: financial apps don't need to be cluttered. This prototype explores what a neobank feels like when you strip everything down to what matters.

Fork it. Break it. Make it yours.

## Stack

- **Framework** — NextJS 16 (App Router)
- **Styling** — TailwindCSS
- **Linter** - BiomeJS
- **Package manager** — pnpm

## Getting Started
```bash
pnpm install
pnpm run dev
```

Open [http://localhost:3000](http://localhost:3000).
